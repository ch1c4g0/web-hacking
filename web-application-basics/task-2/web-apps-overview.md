
<p><h1>The Front End</h1></p>
<p>This is where users interact with the application, HTML, CSS, and Java Script make this possible.</p>

<p><h3>HTML</h3></p>
<p>HTML(Hypertext Markup Language) is the foundation of web applications.<br>
HTML serves as instructions of what to display and how to display it.</p>

<p><h3>CSS</h3></p>
<p>CSS (Cascading Style Sheets) is responsible for standard appearence such as colors, style, and layout.</p>

<p><h3>Java Script</h3></p>
<p>Otherwise known as JS, allows for more complex instructions compared to HTML.</p>

<p><h1>The Back End</h1></p>
<p>These are the things the interacting user does not see but serve a critical function<br>
to how data is stored, processed, and understood.</p>

<p><h3>A Database</h3></p>
<p>This is where data is stored, modified, and retrieved. </p>

<p><h3>Infrastructure</h3></p>
<p>This refers to the architecture the web application is running on.<br>
(Web servers, Application servers, Storage, and Networking Components.</p>

<p><h3>Web Application Firewall (WAF)</h3></p>
<p>Helps filter out harmful requests to the Web Application</p>
